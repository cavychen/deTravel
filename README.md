# 萊茵河谷 Workation 手帳報告 (Rhine Valley Workation Journal)

這是一個單頁面靜態網頁（Single Page HTML），用來記錄四人團隊前往德國萊茵河谷進行 Workation（工作與度假結合）的 10 日專屬手帳報告。專案內包含了每日詳細行程動線、分工紀錄、專屬的網頁視覺排版，以及清晰的團體住宿與交通預算對比表。

## ✨ 專案特色

*   **🎨 純前端打造**：僅使用單一 HTML 檔案，無需安裝 node.js 或設定複雜的開發環境。
*   **⚡ 響應式設計 (RWD)**：透過 CDN 引入 Tailwind CSS 快速建構版面，無論是用手機、平板還是電腦螢幕瀏覽，都能保持絕佳的閱讀體驗。
*   **🖌️ 客製化視覺特效**：包含拍立得相框（Polaroid）、手寫風格引言區塊、斜紋印章標籤以及信紙網格背景，營造出真實手帳本的溫度感。
*   **📊 預算與行程表格**：將原計畫與 4 人行新增的住宿費用並列顯示，方便團隊對比查閱與財務結算。
*   **🔤 質感字體**：整合 Google Fonts 的 Noto Sans TC 與 Noto Serif TC，讓繁體中文字體呈現更具質感。

## 🛠️ 使用技術

*   **HTML5**：網頁骨架與語意化標籤。
*   **Tailwind CSS (via CDN)**：快速且強大的 Utility-first CSS 框架，負責全站佈局與基礎樣式。
*   **Custom CSS**：處理特殊的手帳特效（如旋轉角度、特殊陰影、網格背景等）。
*   **FontAwesome (via CDN)**：提供具指示性的向量圖示（Icons）。
*   **Google Fonts**：提供繁體中文網頁字型。

## 🚀 如何執行與預覽

1. 將最新的程式碼複製並儲存為 `deT42.html`。
2. 確認您的電腦有連上網路（因為需要透過 CDN 載入 Tailwind CSS、字體與圖片）。
3. 對著 `index.html` 點擊兩下，使用任何現代瀏覽器（如 Google Chrome、Microsoft Edge、Safari 等）開啟即可直接預覽完整網頁。

## 📝 內容修改指南

若未來需要微調行程或費用，您可以直接使用任何文字編輯器（如 VS Code、Sublime Text 或記事本）打開 `index.html` 進行編輯：

*   **修改行程文字**：搜尋 `<!-- 7/xx -->` 區塊，直接修改 `<p>` 或 `<h3>` 標籤內的文字內容。
*   **替換照片**：搜尋對應的 `<img src="...">`，將 `src` 裡面的網址替換成您的新照片連結即可。
*   **修改預算**：搜尋 `<!-- 3. 預算結算 -->`，在 `<table>` 區塊內更改費用數字。
